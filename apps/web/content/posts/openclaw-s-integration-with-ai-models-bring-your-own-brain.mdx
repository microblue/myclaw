## OpenClaw's Integration with AI Models: Bring Your Own Brain

OpenClaw's superpower isn't what it does. It's what model you put inside it.

Most AI services lock you into one model. OpenClaw lets you choose.

## Why This Matters

Different models are good at different things:

```
Claude (Anthropic): Best at reasoning, careful, thoughtful
GPT-4 (OpenAI): Best at code and creativity
Gemini (Google): Best at multimodal (images, video, text)
Llama (Meta): Best local option, privacy-first
Mistral: Lightweight, efficient
Grok: Newer, unrestricted
```

Lock yourself into one model? You miss the others' strengths.

## How OpenClaw Model Selection Works

### Configuration

```javascript
// Use Claude
{
  "model": {
    "provider": "anthropic",
    "model": "claude-3-5-sonnet",
    "api_key": "sk-ant-..."
  }
}

// Use GPT-4
{
  "model": {
    "provider": "openai",
    "model": "gpt-4-turbo",
    "api_key": "sk-..."
  }
}

// Use Gemini
{
  "model": {
    "provider": "google",
    "model": "gemini-2.0-pro",
    "api_key": "..."
  }
}

// Use local Llama
{
  "model": {
    "provider": "ollama",
    "model": "llama2",
    "endpoint": "http://localhost:11434"
  }
}
```

One line change. Different brain.

### Switching Models at Runtime

No need to restart:

```bash
# Currently using Claude
npm run agent:switch gpt-4

# Now using GPT-4
npm run agent:switch gemini-pro

# Back to Claude
npm run agent:switch claude
```

Instant switch. Same agent. Different capabilities.

## Choosing the Right Model for the Job

### For General Tasks

**Best:** Claude 3.5 Sonnet

- Balanced reasoning and speed
- Excellent at following complex instructions
- Good at creative tasks
- Thoughtful responses

### For Code

**Best:** GPT-4 or Claude 3.5

- Both excellent at code generation
- GPT-4 slightly better at novel algorithms
- Claude better at explaining code

### For Images/Multimodal

**Best:** Gemini or GPT-4 Vision

- Analyze images
- Extract text from images
- Understand visual context

### For Privacy/Local

**Best:** Llama 2 or Mistral

- Runs locally
- No external API calls
- Complete privacy
- Slightly lower capability (trade-off)

### For Speed/Cost

**Best:** Mistral or GPT-3.5

- Fast response times
- Cheap API costs
- Good for high-volume tasks

### For Unrestricted Reasoning

**Best:** Grok

- Fewer guardrails
- Good for research
- Less "helpful" filtering

## Cost Optimization

Different models cost differently:

```
Per 1M tokens:

Claude 3.5 Sonnet: ~$3-15
GPT-4 Turbo: ~$10-30
Gemini Pro: ~$0.50-2
Llama 2 (local): $0
Mistral: ~$0.50-2
```

For high-volume tasks, switch to cheaper model:

```bash
# Task 1: Complex reasoning (use expensive Claude)
npm run agent:switch claude
npm run task "Analyze this complex legal document"

# Task 2: Simple transformation (use cheap Gemini)
npm run agent:switch gemini-pro
npm run task "Convert CSV to JSON for 10,000 records"

Result: Better capabilities + lower costs
```

## Custom Model Integration

Use proprietary models:

```javascript
// Register custom model
OpenClaw.registerModel({
  name: 'custom-llm',
  endpoint: 'https://your-api.com/v1/models/chat',
  apiKey: 'your-api-key',
  capabilities: ['text', 'reasoning', 'function-calling']
});

// Use it
npm run agent:switch custom-llm
```

Your model. Your control.

## Model Fallbacks

Configure fallback chain:

```javascript
{
  "models": [
    {
      "primary": "gpt-4",
      "fallback": "claude",
      "fallback": "gemini"
    }
  ]
}
```

If GPT-4 is down, automatically use Claude. Then Gemini. Automatic failover.
